Javascript 자료형(Javascript Data Types)

숫자형, TS에서는 'number'로 표기

JavaScript는 정수를 위한 런타임 값을 별도로 가지지 않으므로, int 또는 float과 같은 것은 존재하지 않습니다. 모든 수는 단순히 number입니다

문자열, TS에서는 'string'으로 표기

"Hello, world"와 같은 문자열 값

배열, TS에서는 []로 표기

[1, 2, 3]과 같은 배열의 타입을 지정할 때 number[] 구문을 사용

문자열일 경우 string[](Array<string>으로도 표기 가능(제네릭 개념 이해 필요))
